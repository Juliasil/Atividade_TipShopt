# Loja TipShopt

Este é o repositório para o projeto Loja TipShopt. Este projeto é uma loja online fictícia com um design simples e responsivo.

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript

## Estrutura de Arquivos

- **index.html**: Página principal da loja.
- **style.css**: Estilos para o projeto.
- **script.js**: Script JavaScript para interações na página.
- **img**: Diretório contendo imagens utilizadas no projeto.

## Instruções de Uso

1. Clone este repositório: `git clone [URL do Repositório]`.
2. Abra o arquivo `index.html` em seu navegador para visualizar a loja.
